Architecture

Microkernel by design, not by slogan.

Mach Core

IPC, VM, tasks, threads, scheduler. Keep authority narrow and interfaces sharp.

System Servers

BSD, VFS, network, device, auth in userspace. Crash isolation and replaceable components.

Framework Layer

Foundation, AppKit, libdispatch, CoreFoundation, and Objective-C runtime integrations.

DPS-inspired display server architecture aimed at a NeXT-heritage desktop experience.
