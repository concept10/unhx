# archive/cmu-mach/

CMU Mach 3.0 reference archive.

## Overview

The original Carnegie Mellon University Mach 3.0 kernel — the primary design
reference for UNHOX. Developed 1985–1994 by Richard Rashid, Avie Tevanian,
and colleagues.

## Source Locations

- Utah Flux archives: http://www.cs.utah.edu/flux/mach4/
- Bitsavers: https://bitsavers.org/bits/CMU/
- GNU mirrors of CMU Mach 3.0
