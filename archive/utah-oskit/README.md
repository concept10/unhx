# archive/utah-oskit/

Utah OSKit and Lites reference archive.

## Overview

The University of Utah Flux project produced two important artifacts:

### OSKit

A component-based OS construction kit — provides standalone C library, device
drivers, and OS primitives as composable libraries. Extremely useful for early
UNHU bring-up without needing a complete OS stack.

### Lites

A complete BSD 4.4 server running on Mach 3.0 — the most complete historical
reference for what a BSD personality server on Mach looks like.

## Source Locations

- Utah Flux archives: http://www.cs.utah.edu/flux/oskit/
- Lites: http://www.cs.utah.edu/flux/lites/
